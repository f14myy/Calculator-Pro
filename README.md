# Calculator Pro 🤑

Мой первый шуточный проект на Svelte. Это вроде бы обычный калькулятор, но с одним нюансом: за все базовые функции нужно платить. 

### В чём суть?
- Бесплатно можно складывать числа и пользоваться плюсом.
- За попытку вычесть, умножить, поделить или включить тёмную тему вас попросят купить подписку PRO или ULTRA.
- При оплате подписки просит решить "очень сложную капчу", а потом кидает на Рикролл.
- Кнопка `=` заставляет калькулятор думать 10 секунд и в итоге выдает "Hello world" вместо ответа. (А ещё намекает, что если купить подписку, калькулятор будет думать быстрее).

### Как запустить у себя
Если вдруг хотите поковырять код локально:

1. Склонируйте репу
2. Установите зависимости (я юзаю pnpm, но можно и npm):
   ```bash
   pnpm install
   ```
3. Запустите в дев-режиме:
   ```bash
   pnpm dev
   ```
